# Сэтгэлийн илгээмж – Romantic Greeting Web App

Тусгай романтик мэндчилгээний веб хуудас үүсгэх захиалгын систем ба хүлээн авагчид зориулсан хувийн веб хуудас.

## Үнэлгээний түвшин

| Түвшин | Үнэ | Онцлог |
|--------|-----|--------|
| **Үнэгүй – Урьдчилан харах** | 0₮ | Системийн мессеж, хүлээн авагчийн нэр, хязгаарлагдмал холбоос. Зураг/видео/дуу байхгүй. |
| **Стандарт – Зургийн багц** | 5,000₮ | Хувийн веб, 1–5 зураг, урьд бичсэн эсвэл богино мессеж (200 тэмдэгт). |
| **Премиум – Дээд туршлага** | 20,000₮ | Стандартын бүх зүйл + видео мессеж (MP4, 30–90 сек), романтик дуу (авто тоглуулах, унтраалттай), урт мессеж. |
| **AI – Хиймэл оюун ухаан** | 30,000₮ | Премиум бүх онцлог. Зураг/видео оруулахгүй – AI-ийн романтик зургууд, видео, мессеж, дуу. Сэтгэл хөдлөл (Романтик/Сэтгэлтэй/Сэтгэл хөдлөм/Элегант), түлхүүр үгс, мессежийн аялгуу (романтик/сэтгэл хөдлөм/шүлгэн). |

## Онцлог функцууд

- **Emotional Reveal** (Премиум/AI): «Нээх» товч дарснаар бяцхан нууц, зүрх анимаци, дуу эхэлнэ.
- **Hidden Love Note** (Премиум/AI): Нууц од/одод дарвал «Та үүнийг оллоо. Би чамайг үнэхээр хайрладаг.» гэсэн мессеж гарна.
- **Mood Switch** (Премиум/AI): Өдөр/Шөнийн горим солих товч.
- **Voice Love Message** (Премиум/AI): Дуу мессеж тоглуулах.
- **Handwritten Animation** (Стандарт/Премиум/AI): Мессеж үсэг үсгээр бичигдэж байгаа мэт харагдана.
- **AI Love Story** (AI): «Бидний түүх» хэсэг – AI-ийн хайрласан түүх.
- **Time Capsule** (Премиум/AI): «6 сарын дараа нээгдэнэ» гэсэн мессеж.
- **Ending Scene** (бүх төлбөртэй): «Энэ нь зөвхөн эхлэл…» болон «Хайртайгаар, {{senderName}}».

## Ашиглах

1. **index.html** нээнэ.
2. **Загвараа сонгоно уу**: Үнэгүй, Стандарт (5,000₮), Премиум (20,000₮), эсвэл AI (30,000₮).
3. **Үнэгүй**: Нэр оруулаад **Урьдчилан харах** дарна.
4. **Стандарт/Премиум/AI**: Нэр, илгээгчийн нэр, холбоо барих бөглөнө. Премиум/AI: дуу мессеж, цаг хугацааны капсул. AI: хайрласан түүх (хэзээ уулзсан, юунд дуртай, ирээдүйд юу хүсэж байна).
5. **Төлбөрийн заавар харах** дарж шилжүүлэх заавар харна.
6. Төлбөрийн баримтаа оруулж **Баталгаажуулах** дарна.
7. Амжилтын дэлгэцэнд **Веб рүү орох** эсвэл **Хуваалцах** хийнэ.

Хүлээн авагч холбоос нээхэд `greeting.html` хуудсан дээр бүх онцлогууд харагдана.

## Backend / Frontend

**Frontend only** (sessionStorage): index.html нээгээд ашиглана. Нэр, мессеж, зураг нь ижил цонхонд "Веб рүү орох" дарсны дараа харагдана. Холбоос хуваалцахад бэкенд шаардлагатай.

**Backend + Frontend** (холбоос хуваалцах боломжтой):
1. `АЖИЛЛУУЛАХ-БЭКЕНД.bat` дарж backend эхлүүлнэ (http://localhost:3000)
2. `index.html` нээнэ (эсвэл `АЖИЛЛУУЛАХ.bat`)
3. Захиалга өгөж, төлбөр баталгаажуулна
4. Гарсан холбоосыг хуваалцахад нэр, зураг, мессеж бүгд харагдана
